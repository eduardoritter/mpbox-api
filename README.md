
#mpbox-api

## Description
A [Nest](https://github.com/nestjs/nest) example API using [TypeORM](https://github.com/typeorm/typeorm) with SQLite for simplicity.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
